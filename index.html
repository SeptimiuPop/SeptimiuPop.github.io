Academic Projects:

Chess Society (Website, LAMP stack)

This was a lab project that I worked on, with on-the-job training. It is a web application for maintaining an imaginary chess society, hosting numerous users.

London Property Manager (Java/JavaFX)

This was a group project, created as an application for exploring rental properties using existing data.

Kanban Board Manager (Java/JavaSWING)

As a team of 5 people we developed a desktop clone for Trello using agile methodologies. The app works as a kanban board, allowing users all the nececssary capabilities to create, edit and maintain task tickets. Taking after Trello, we also provided our users with data analytics for better tracking and ticket priority systems.

Pacman AI (Python)

Over a few diferent courses in university I had projects that involved creating smart agents that take control over the player character in order to win the game. The algorithms used involved model-based solutions like value iteration as well as model-free solutions like q-learning and SARSA. On a separate project I was also tasked with gathering data from the game and creating a classifier that can distinguishes between good and bad moves.

Path Searching Algorithms (Java/Processing)

As part of a robotics project I was taked with implemented various path search algorithms. The goal of this project was to guide a virtual robot to it's escape form randomly generated mazes. Some of the algorithms that I chose include A*, Bug Search and Wavefront.
